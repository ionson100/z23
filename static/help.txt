Генерация шаблона документа
Запускаем программу в нужной директории с правами на запись
В директории создается папка static с подключаемыми ресурсами
Создается папка content для содержания картинок документа.
Создается файл шаблона tenplateindex.txt ( шаблон документа)
Создается index.html этого шаблона.
Запускается на браузере по умолчанию.
